Điều khoản sử dụng tính năng AI

Trước khi tiếp tục, vui lòng đọc và đồng ý với các điều khoản sau:

  • Bạn sẽ sử dụng API Key Google Gemini của riêng bạn. Mọi chi phí phát sinh (nếu có) từ việc sử dụng API này là trách nhiệm của bạn.
  • Trang web này chỉ đóng vai trò là giao diện để gửi yêu cầu và không lưu trữ API Key của bạn. Key của bạn chỉ được gửi trực tiếp đến Google.
  • Không nhập thông tin nhạy cảm hoặc cá nhân vào ô câu hỏi.

Bằng cách nhấn "Tôi đồng ý", bạn xác nhận đã đọc, hiểu và chấp nhận các điều khoản trên.

Không có API Key? Xem hướng dẫn lấy key tại đây.

Hướng dẫn lấy Google Gemini API Key (Miễn phí)

  1. Truy cập trang Google AI Studio tại: https://aistudio.google.com/app/apikey
  2. Đăng nhập bằng tài khoản Google của bạn.
  3. Nhấn vào nút "Get API key".
  4. Bạn có thể cần tạo một dự án mới (Create API key), sau đó nhập tên dự án và chọn loại dự án.
  5. Đợi vài giây, sau đó sẽ ấn vào API vừa hiện ra.
  6. API Key của bạn sẽ hiện ra. Nó là một chuỗi ký tự dài. Hãy sao chép (copy) toàn bộ chuỗi ký tự này.
  7. Quay lại trang này và dán (paste) key vào ô "API Key Gemini của bạn" ở trên.

Lưu ý: Hãy bảo mật API Key của bạn và không chia sẻ nó công khai.
